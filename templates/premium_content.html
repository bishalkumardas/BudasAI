🎬 YouTube Creator Workflow
Full Video Production Pipeline
From blank page to published video with repurposed social content — entirely AI-assisted. Saves a solo creator 8–12 hours per video.
⏱ ~35 mins total 🤖 Claude + Canva AI 📹 Works for any niche 🔁 Repeatable every week
Phase 1 — Pre-Production
01
Topic & Angle Research
🤖 Claude ⏱ 5 mins
▾

Before writing a single word, generate 20 validated video ideas ranked by search potential. Claude acts as your research assistant — paste this prompt and replace the placeholders.

Prompt — copy & paste into Claude
You are a YouTube strategy expert. My channel is about [YOUR NICHE] and my target audience is [e.g. beginner freelancers aged 20-30]. Generate 20 YouTube video ideas. For each idea provide: - A click-worthy title (under 60 characters) - The search intent behind it (informational / problem-solving / entertainment) - Estimated monthly search volume: High / Medium / Low - One competitor gap — why this hasn't been done well yet Rank them from highest to lowest potential. Output as a clean table.
💡 Pro tip: After Claude gives you the list, paste the top 5 titles into Google Trends and compare. Pick the one with an upward trend, not a peak.
02
Full Script Writing
🤖 Claude ⏱ 10 mins
▾

Paste your chosen topic. Claude will write a complete, structured script with hook, sections, B-roll cues and CTA — ready to record.

Prompt — copy & paste into Claude
Write a YouTube video script on: [YOUR TOPIC] Channel details: - My tone: [e.g. conversational, direct, slightly humorous] - Target audience: [e.g. small business owners who are new to AI] - Video length target: [e.g. 8-10 minutes] Script structure required: 1. HOOK (first 30 seconds) — open with a bold statement or surprising fact 2. INTRO — who this video is for and what they'll learn 3. MAIN SECTIONS (3-5 sections with clear subheadings) 4. B-ROLL CUES — in brackets after relevant sentences e.g. [B-roll: screen recording of tool] 5. TRANSITIONS — smooth sentences linking each section 6. CALL TO ACTION — subscribe + comment prompt 7. OUTRO — 15 seconds max Do not use filler phrases like "In today's video" or "Without further ado". Start strong.
✅ What you get
A word-for-word script (~1,500 words for 8-min video), B-roll directions, section headers you can use as chapter timestamps, and a CTA that doesn't feel forced.
03
Title, Description & Tags (SEO)
🤖 Claude ⏱ 3 mins
▾

YouTube SEO is 40% of discoverability. This prompt generates everything you need for the upload page in one shot.

Prompt — copy & paste into Claude
I'm uploading a YouTube video about: [TOPIC] Target keyword: [MAIN KEYWORD e.g. "AI tools for students"] Generate: 1. 10 title options — mix of curiosity, how-to, list, and controversy formats. Keep each under 60 characters. Flag the top 3. 2. YouTube description — 250 words, keyword-rich, natural tone. Include: 1-line hook, 3-sentence summary, chapter timestamps (placeholder format 0:00, 2:30 etc), 3 relevant links as [LINK 1], CTA to subscribe. 3. 30 tags — mix of broad, niche, and long-tail. Comma-separated, no hashtags.
Phase 2 — Post-Production & Repurposing
04
Thumbnail Brief (Canva AI)
🤖 Claude + Canva AI ⏱ 5 mins
▾

Claude writes the thumbnail brief, Canva AI creates the visual. No designer needed.

Prompt — copy & paste into Claude
Write a YouTube thumbnail design brief for this video: [TOPIC] Include: - Text overlay: max 4 words, high contrast, what should it say? - Colour palette: 2-3 hex codes that create high click-through contrast - Emotion/facial expression: if a face is used, what should it convey? - Background style: solid, gradient, scene — describe it - Composition: where should text, face and visual elements be placed? - Canva AI image prompt: a ready-to-paste prompt for Canva's AI image tool
💡 Pro tip: Copy the Canva AI image prompt directly from Claude's output and paste it into Canva → Elements → AI Image Generator.
05
Auto-Repurpose to Instagram + LinkedIn + X
🤖 Claude ⏱ 5 mins
▾

Paste your script or transcript — Claude converts it into platform-native content for every channel in one go.

Prompt — copy & paste into Claude
Here is my YouTube video script/transcript: [PASTE YOUR SCRIPT HERE] Repurpose this into: 1. Instagram caption (max 150 words): hook in first line, value in body, CTA at end, 5 relevant hashtags 2. LinkedIn post (200-250 words): professional tone, story-led, no cringe buzzwords, end with a question 3. X / Twitter thread: 6-8 tweets, each standalone, first tweet is the hook, last tweet links back to YouTube 4. Email newsletter intro (100 words): conversational, as if writing to a friend, include YouTube link placeholder
✅ What you get
4 pieces of platform-native content from one script. Total time: 5 minutes copy-paste. Without AI this would take 2+ hours manually.

30-Day Content Calendar in 30 Minutes

⏱ ~30 mins total🤖 Claude + Canva AI📅 Full month planned
Phase 1 — Strategy & Planning
01
30-Day Content Calendar
🤖 Claude⏱ 5 mins
▾

One prompt, one full month of content ideas. Claude creates a balanced mix of content types so your feed never feels repetitive.

Prompt — copy & paste into Claude
Create a 30-day Instagram content calendar for my account. My niche: [e.g. personal finance for Gen Z] My audience: [e.g. 18-25 year olds who want to save money] My goal: [e.g. grow followers + drive DMs] Posting frequency: [e.g. 5x per week] For each post include: - Day number and content type (Reel / Carousel / Static / Story) - Post topic / hook - Content angle (educational / entertaining / personal story / promotional) - Suggested caption style (short punchy / long storytelling / question-based) Use a 4:1 ratio — 4 value posts for every 1 promotional post. Output as a table.
02
Carousel Post — 10-Slide Script
🤖 Claude⏱ 5 mins
▾

Carousels get 3× more saves than regular posts. This prompt gives you a complete slide-by-slide script ready to drop into Canva.

Prompt — copy & paste into Claude
Write a 10-slide Instagram carousel post on: [TOPIC] Audience: [YOUR AUDIENCE] Tone: [e.g. bold and direct / warm and encouraging] Slide 1: Cover — must stop the scroll, make it controversial or bold Slides 2-9: One insight per slide, build on each other Slide 10: CTA — tell them exactly what to do next For each slide: headline (max 7 words) + body copy (max 25 words) + visual direction Also write: caption (150 words max) + 10 hashtags
03
Reels Script (Viral Format)
🤖 Claude⏱ 4 mins
▾

Reels under 30 seconds get the highest reach. This prompt writes a punchy, hook-first script in the exact format that Instagram's algorithm rewards.

Prompt — copy & paste into Claude

No "Hey guys". - 3-20 sec BODY: 3 rapid-fire points, each 1-2 sentences max. - 20-28 sec PAYOFF: The "aha moment" — surprising fact or result - 28-30 sec CTA: One action — Save this / Follow / Comment your answer. Also provide: on-screen text overlays + background suggestion + audio style

Mini tutorial (150 words): step-by-step instructions on [SPECIFIC TASK] 5. Social proof (100 words): frame a result or achievement naturally, CTA at end Topics: [LIST YOUR 5 TOPICS] | Voice: [e.g. confident, relatable] Add 5 relevant hashtags at the end of each caption.

📊 Data Analyst Workflow
Analyse Faster. Report Smarter.
Use Claude to clean data, write SQL queries, explain findings in plain English and turn raw numbers into executive-ready reports.
⏱ ~40 mins end-to-end🤖 Claude + Excel / Python📊 Works for any dataset
Phase 1 — Data Preparation
01
Clean & Understand Your Dataset
🤖 Claude⏱ 5 mins
▾

Paste your raw CSV headers or a sample — Claude audits data quality, flags issues and suggests the right analysis approach.

Prompt — copy & paste into Claude
I have a dataset with these columns: [PASTE YOUR CSV HEADERS OR FIRST 5 ROWS HERE] Business context: [e.g. monthly sales data for an e-commerce store] Please: 1. Describe what this dataset contains and what each column likely represents 2. Flag data quality issues to fix before analysing (nulls, duplicates, type mismatches) 3. Suggest 5 high-value analysis questions this dataset can answer 4. Recommend the best tool: Excel / Python (Pandas) / SQL — and why
💡Pro tip: Upload the file directly to Claude if it's under 10MB — it reads actual data and gives far more specific insights.
02
Write SQL or Python Code On Demand
🤖 Claude⏱ 3 mins
▾

Describe what you want to find out in plain English — Claude writes the exact SQL or Python/Pandas code.

Prompt — copy & paste into Claude
My table/dataframe: [TABLE NAME] Columns: [LIST KEY COLUMNS] Tool: [PostgreSQL / MySQL / Python Pandas / Excel] Write code to: [PLAIN ENGLISH — e.g. "Find top 10 customers by revenue in last 90 days, broken down by product category"] Requirements: - Comment each step - Handle nulls gracefully - Show expected output format
✅ What you get
Production-ready SQL or Python with comments, null handling and expected output — ready to run immediately.

Explain Your Results in Plain English

Summarise the key finding in one sentence 2. Explain what the numbers mean in plain business English 3. Highlight 3 specific insights worth acting on 4. Flag anything unusual or worth investigating 5. Suggest one follow-up analysis question
04
Write the Executive Report
🤖 Claude⏱ 8 mins
▾

Turn raw findings into a polished, structured report ready for management.

Prompt — copy & paste into Claude
Write a data analysis report based on: [PASTE KEY NUMBERS, INSIGHTS AND CONCLUSIONS] Title: [e.g. "Q3 2024 Sales Performance Analysis"] Audience: [e.g. senior management, non-technical] | Tone: [formal and concise] Structure: 1. Executive Summary — 3 sentences, key finding + recommendation 2. Key Metrics — 4-5 numbers that matter, with context 3. What The Data Shows — 3-5 plain-English insights 4. Anomalies & Watch Points 5. Recommendations — 2-3 specific next steps Under 400 words. No jargon.
💡Pro tip: After the report, ask: "Now write a 3-bullet TL;DR for a Slack message."
